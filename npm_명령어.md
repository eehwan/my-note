# npm 기본 사용법
<br>
<br>

```
npm install @module_name@
npm install @module_name@ -g
npm install @module_name@ -d
```
#### 모듈을 설치한다 default옵션은 -s
------------------------------
```
npm list -g --depth=0
```
#### 하위모듈을 제외한 설치된 모듈을 볼 수 있다
--------------------------------------
```
npm uninstall @module_name
```
#### 모듈을 삭제한다
