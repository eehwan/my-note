# Pythonic Code

#### list()
```
list("abc") => ["a", "b", "c"]
```
#### split()
```
"abc".split("b") => "a", "c"
```
#### zip()
```
list_a = [1, 2, 3]
list_b = [3, 2, 1]

zip(list_a, list_b) => (1,3), (2,2), (3,1)
```
#### enumerate()
```
new_list = [4, 3, 3]
enumerate(new_list) => (0,4), (1,3), (2,3)
```
